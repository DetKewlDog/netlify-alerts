Netlify Alerts



Use this tool to enable Desktop push notifications when Netlify deploys your website!
Click on Generate Webhook to get your webhook URL.
In Netlify, navigate to Site Configuration > Build & Deploy > Deploy notifications.
Choose Add notification > Outgoing webhook.
Choose Deploy succeeded as the event to listen for.
Paste the Webhook URL as the URL to notify.
Save the webhook and you're done!
